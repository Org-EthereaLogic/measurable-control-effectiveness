# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly by emailing **anthony@etherealogic.com**.

Please do not open a public issue for security vulnerabilities.

## Scope

This repository contains a benchmark for data quality controls. It does not
process real customer data or connect to production systems.

|---------|-----------|
